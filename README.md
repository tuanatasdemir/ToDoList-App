# To-Do List Application

A modern desktop To-Do List application built with C# and .NET Windows Forms.
It uses **SQLite** for database management to ensure portability and data persistence.

## Features
- **Add/Delete Tasks:** Easily manage daily tasks.
- **Sound Effects:** Custom audio feedback for actions.
- **Persistent Data:** Uses SQLite so your tasks are saved even after closing the app.
- **Modern UI:** Clean, flat design interface.
- **Portable:** No installation required.
